## 1. この案件専用のデザインコンセプト
### キーワード： 「現場の確かさと、変わらぬ信頼。― FACILITY, MANAGED.」
ビルメンテナンスは“見えないところで建物を整え続ける”仕事。だから訴えるのは派手な清涼感ではなく、
**規律・体系・人の手による確かさ**。冷たいガラスの清潔感（前案の方向）ではなく、
**紺の堅実さ × 真鍮（ブラス）の温度** で「きちんと管理され、長く付き合える会社」を視覚化する。
建物のフロア／区画を想起させる**モジュラー・グリッド**を全面のモチーフに据え、
“整然と管理されている”状態そのものをデザイン言語にする。
→ 過去ITLINE実績とも、報告書のグリーン案とも、前バージョンのアクア清潔感とも異なる固有トーン。

## 2. 配色パレット（現行＝紺を主役に。差し色を「アクア」から「ブラス」へ転換し差別化）
| 用途 | 色名 | HEX | なぜこの色か（業種連想・心理） |
|---|---|---|---|
| Primary | Deep Navy | `#13314F` | ロゴの紺を主役に。堅実・信頼・BtoBの安心感（client必須制約） |
| Secondary | Steel Blue | `#3E6694` | 面で使う中間ブルー。規律・管理の体系感。紺一色の重さを抜く |
| Accent | Warm Brass | `#C08A3E` | **前案アクアからの最大の転換点**。真鍮/琥珀＝人の手の温度・確かさ・信頼の重み。CTA・数値・キーラインに少量 |
| Base | Warm Paper | `#F4F1EA` | 冷白ではなく温かみのあるオフホワイト。落ち着き・誠実さ（前案 Cool White と差別化） |
| Text | Ink | `#1F262E` | 墨寄りの濃グレー。長文可読性とブランドの硬質感 |
| 補助線/微 | Mist Gray | `#C7CFD8` | 罫線・区画線・キャプション。グリッドモチーフの細線に |

- **緑は不使用**（client指示=現行ブルー踏襲、報告書グリーン案は不採用）。**アクア(シアン)も使わない**＝前案との明確な差別化点。
- 差し色 Warm Brass は“効かせる”分量に限定（CTA・実績数値・見出し下のキーライン）。無難な青一辺倒に逃げず、紺×ブラスの引き締まった大人のBtoB配色に。

## 3. フォント（Zen Kaku Gothic New を廃し、インフラ／硬派系へ）
Google Fonts 実フォント名：
- **見出し用： `IBM Plex Sans JP`**（700）― 直線的で工業的な端正さ。グローバル企業フォント由来の規律感が「施設管理＝インフラの信頼」に合致。前案の Zen Kaku Gothic New（柔らかい幾何ゴシック）と表情を明確に変える。
- **本文用： `Murecho`**（400/500）― 中立で可読性が高く、わずかに人間味のある現代ゴシック。見出しの硬さに対し本文で温度を補う（前案 BIZ UDPGothic と差別化）。
- **欧文アクセント： `Space Grotesk`**（500/700）― グリッド感のある工業欧文。"FACILITY 01" 等の番号・英語ラベル・数値に。前案 Outfit より硬質。
```html
<link href="https://fonts.googleapis.com/css2?family=IBM+Plex+Sans+JP:wght@400;700&family=Murecho:wght@400;500;700&family=Space+Grotesk:wght@500;700&display=swap" rel="stylesheet">
```

## 4. 世界観・レイアウト方向性
- **トンマナ**: 規律・体系・確かさ（硬派だが冷たくない＝紺×ブラスの温度で中庸を担保）。直線基調、薄い区画線、整然としたブロック割。
- **装飾モチーフ（1つに統一）**: **モジュラー・グリッド／区画線**。建物のフロア・区画図を想起させる細いMist Gray罫線と、要所に1pxのBrassキーライン。背景にうっすら方眼。曲線・斜め・手書きは使わない。
- **③ ヒーローの型＝D：非対称グリッド（大小ブロック）**【前案=B全面写真オーバーレイ → 明確に変更】
  - 左に大きな Deep Navy ブロック：見出し「現場の確かさと、変わらぬ信頼。」＋英語サブ "FACILITY MANAGEMENT SINCE 2007"＋Brass の細CTA。
  - 右に小ブロックを格子状に：①清掃/衛生/設備への3つの入口タイル ②実績数値タイル（創業年・対応施設数） ③管理された空間の写真タイル1枚。
  - → “整然と区画されたグリッド”自体が「体系的に管理する会社」を物語る。単一の大写真に頼らず個性を出す。
- **個性の出し方（CSS）**: ①Brassの1pxキーライン ②区画線グリッドの背景 ③数値カウンター（創業2007〜の年数・対応業務3領域） ④清掃01/衛生02/設備03のナンバリングをグリッドに整列 ⑤カードはホバーで枠線がBrassに変わる（持ち上げ影は控えめ＝硬派さ維持）。
- **セクション構成と順序（TOP）**＝BtoB施設管理の意思決定導線「信頼 → 領域理解 → 実務の裏付け → 相談」。定番の量産レシピに流されない：
  1. ヒーロー（誰向け・何の会社か即伝達／グリッドで世界観提示）
  2. 私たちが選ばれる理由 ＝ 3つの管理姿勢（信頼の根拠を先に）
  3. 事業領域 3業務（清掃管理 / 衛生管理 / 設備管理）をナンバリングのグリッドカード
  4. 対応施設・実績（数値＋施設タイプで実務感。現行の更新停止感を払拭）
  5. 管理品質へのこだわり（点検フロー・体制＝“見えない仕事”の可視化）
  6. お知らせ / Topics（継続運用の印象づけ）
  7. 採用バナー（人＝品質の源泉として接続）
  8. CTA帯（電話＋お問い合わせ）／フッター（サイトマップ全展開）

## 5. 参考の選定（方向性のみ・見た目は本案件用に再構築）
- 報告書記載のイオンディライト／日本管財／大成 … **BtoB信頼骨格（事業＋管理品質＋体制＋採用の一貫設計）**を構成参考。
- 過去デザイン `https://www.tpc-cop.co.jp/` … **BtoBコーポレートのグリッド整理・信頼の見せ方**を参考。
- 参考サイト `株式会社進藤組（総合建設）` … **現場系×硬派タイポ／区画的レイアウト**の質感を参考。
- ※配色・フォント・世界観は本企画書の独自指定を優先。前案・他案と同じ見た目にしない。

**採用する参考画像**（業種・トンマナ適合のみ選定。緑系・汎用夕景は除外）:
- ヒーロー右の空間タイル: `MV07_オープンオフィス執務風景_OFFICE-WORK.png` … 清潔に管理された空間。
- 会社/信頼: `MV03_高層会議室と都市眺望_OFFICE-ARCHITECTURE.png` … ブルートーンのBtoB信頼感。
- スタッフ/現場の人: `MV19_ロビーを歩くチーム_PEOPLE-LOBBY.png` or `MV08_ロビーを歩く5人チーム` … 管理を担う人の存在。
- ※**全面1枚写真には依存しない**（前案がそれ）。写真はグリッドのタイルとして小さく使い、主役は紺×ブラス×区画グリッドのCSSビジュアル。清掃/衛生/設備の現場写真は適合素材が無いため CSSアイコン＋プレースホルダで構成（実写差し替え前提）。緑系 MV06/13/16・汎用夕景 MV01 は不採用。

## 6. 作成する下層ページ（クライアント「下層ページまで」明示 → 必ず含む）
